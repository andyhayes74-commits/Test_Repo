# Hayfam Books visual preview

This repository uses Playwright and GitHub Actions to render the live Hayfam Books website in a real Chromium browser.

The workflow captures the homepage and Books catalogue at desktop and mobile sizes. Current screenshots are saved as a workflow artifact named `hayfam-screenshots`.

## Captures

- Home desktop top
- Home desktop full page
- Home mobile top
- Home mobile full page
- Books desktop top
- Books desktop full page
- Books mobile top
- Books mobile full page
